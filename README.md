# SiniorAiBot v5

نسخه ۵ شامل مدیریت مالک، اعلان حرفه‌ای رسید، درخواست و نتیجه، امتیازدهی نتایج ۱ تا ۱۰، نمایش سه نتیجه برتر، علاقه‌مندی قابل تغییر و چیدمان جدید دکمه‌هاست.

## تنظیمات مهم

فایل `.env.example` را به `.env` تغییر نام بده و حتماً مقدار `OWNER_ID` را با آیدی عددی تلگرام مالک پر کن.

```env
OWNER_ID=94112865
ADMIN_IDS=
```

مالک محدودیت پرامپت، آموزش و جوین اجباری ندارد و پنل مدیریت را می‌بیند.

## اجرا

```bash
npm install
npm run dev
```
